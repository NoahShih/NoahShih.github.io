HTML Reference Page


HTML

The HTML start tag is written as <html> and it is the second element written after <!DOCTYPE html>. The end tag is written at the end with a black slash in front. This html element shows the start and end of the html format.

An example would look like this:

<!DOCTYPE html>

<html>

</html>

Head

The Header start tag is written as <head>. It is the element written after the html start tag, <html> and it is indented. The end tag is written near the end with a black slash in front and another ident. Some people may put style code in this section or element.

An example would look like this:

<!DOCTYPE html>

<html>

<head>

<head>

<html>

Headings

The Headings start tags are written as <h1>, <h2>, <h3>, <h4>, <h5>, <h6>. This element is mainly used in the body. Without using any style code, the heading number will increase as the size of the header decreases. However, you can specifically change the size of a certain header font with style (CSS) code.

An example would look like this:

<body>

<h1></h1>

<h2></h2>

<h3></h3>

<h4></h4>

<h5></h5>

<h6></h6>

<body>

Link to w3schools

Paragraph

The paragraph start tags and end tags are written as <p> and </p>. This element is also mainly used in the body. You can apply style code to the paragraph element which can allow you to do multiple things such as increase the size of the font,change the font, and change the color of the font and text box.

An example would look like this:

<body>

<p></p>

<body>

Breaks

The break tag can be identified as <br>. This element is used to space out text, images, etc. The break element will increase the space between two things by an entire line.

An example would look like this:

<body>

<p></p>

<br>

<p></p>

<body>

Comments

The comment tag is known as <!-- -->. This element is used to make notes that will not affect the code and website. It can also be used to keep code that you will need later, but do not want to use. Basically, you can put anything in the comments element and it will not affect the code.

An example would look like this:

<body>

<!--This will not be seen-->

<body>

Lists

The lists tag is known as <ul> or <ol>. The ul element is used to set up an unordered list. The <ol> is used to create an ordered list. Both list tags are also used with <li> to make lists.

An example would look like this:

<body>

<ul>

<li>Coffee</li>

<li>Tea</li>

<li>Milk</li>

</ul>

<ol>

<li>Coffee</li>

<li>Tea</li>

<li>Milk</li>

</ol>

<body>

Tables

The tables tag is written as <table>. This element is used to make an HTML table. The elements, <tr>,<th>, and <td> are used at least once to make a table. <tr> is used to make a row, <th> is used to make a table header, and <td> defines a standard table cell in HTML.

An example would look like this:

<table>

<tr>

<th>Month</th>

<th>Savings</th>

</tr>

<tr>

<td>January</td>

<td>$1000</td>

</tr>

</table>

Link to w3schools

Blocks

The block tag is written as <div>. This element is used to distinguish a section in HTML. This element is also used to group things together to format them with CSS.

An example would look like this:

<div style="color:#0000FF">

<h3>This is a heading</h3>

<p>This is a paragraph.<p>

</div>
